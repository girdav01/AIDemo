# Docs

Project documentation for the **TrendAI Vision One AI Security Challenge** booth.

- `vision-one-setup/` — **build instructions for the Vision One side of each
  challenge**: what to provision, the synthetic content to plant, the policies to
  configure, and how each maps to the booth app's "clears when" condition. Start
  with `vision-one-setup/00-overview.md`.
- Add your own design docs, scripts, and reference material in this folder.

The booth web app in this repo is a **self-contained simulation** of all eight
challenges (runs offline, no Vision One tenant required). These docs describe how
to stand up the **real Vision One demos** the app mirrors, so the booth can run
live where the product is show-stable and fall back to the simulation otherwise.
